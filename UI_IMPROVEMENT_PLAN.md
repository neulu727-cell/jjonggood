# UI 개선 계획 - 경쟁 앱 분석 기반

## 경쟁 앱 분석 (캡처 기반)

### 1. 캘린더에 예약자 이름 직접 표시 (02, 03번 캡처)
- 날짜 셀 안에 `봉자(비숑...)`, `은동(미...)` 처럼 반려동물이름(견종) 직접 표시
- 3건 이상이면 "+2" 같은 추가 건수 표시
- 색상으로 구분 (보라/연보라 등)
- **현재 우리**: 점(●) 하나로만 표시

### 2. 일일 예약 리스트 뷰 (04번 캡처)
- 시작~종료 시간 명확히 표시: `오전 11:00 ~ 오후 01:00`
- 견종 + 이름 + 보호자이름: `푸들 뚜치 박차희님`
- 🐾 아이콘 + 반려동물이름(견종): `뚜치(푸들)`
- 서비스 경로: `노아네>전체미용`
- 매출 배지 오른쪽에 표시
- **현재 우리**: 타임라인 슬롯 기반 (이미 유사하게 구현됨)

### 3. 고객별 매출 통계 (01번 캡처)
- 구분 / 횟수 / 총액 / 평균 테이블
- 예: 개인 5건, 380,000원, 평균 76,000원
- **현재 우리**: 없음

### 4. 상세 히스토리 탭 (01번 캡처)
- "일반 예약" / "매출" 탭으로 분류
- 각 이력: 날짜/요일/시간, 반려동물(견종), 서비스, 상태배지(예약/완료)
- 예약일시와 완료일시 이중 표기
- 펼치기(>) 버튼으로 상세정보 확인
- **현재 우리**: 고객상세에 이력 있지만 탭 분류/매출 통계 없음

### 5. 미용사/서비스 타입 필터 (02, 03번 캡처)
- 상단 필터 탭: "전체", "미용", "데이케어" 또는 "전체", "원장님"
- 미용사별/서비스별로 캘린더 필터링
- **현재 우리**: 필터 없음

### 6. 검색 + 오늘 버튼 (02, 03번 캡처)
- 상단에 검색(🔍), "오늘" 버튼, 월 드롭다운
- 빠른 네비게이션
- **현재 우리**: 고객검색은 별도 팝업, "오늘" 버튼 없음

## 적용 우선순위 (난이도/효과 기준)

### [1순위] 캘린더에 예약자 이름 표시 ⭐⭐⭐
- 효과: 한눈에 누가 예약했는지 파악 가능
- 난이도: 중
- 방법: MonthCalendar 셀에 예약자 이름 최대 2~3개 표시, 초과시 "+N" 표기
- 컬러 배지로 예약 상태 구분

### [2순위] 고객별 매출 통계 ⭐⭐⭐
- 효과: 고객 가치 파악, 매출 관리
- 난이도: 중
- 방법: CustomerDetail에 매출통계 섹션 추가 (횟수/총액/평균)

### [3순위] 보호자 이름 표시 ⭐⭐
- 효과: 전화 올 때 보호자 이름으로 빠르게 식별
- 난이도: 하 (DB에 name 필드 이미 있음)
- 방법: 타임라인, 수신기록 등에 보호자이름 함께 표시

### [4순위] 예약완료 시간 기록 ⭐⭐
- 효과: 실제 소요시간 파악, 업무 효율 분석
- 난이도: 하
- 방법: 상태 변경 시 completed_at 타임스탬프 기록

### [5순위] 오늘 버튼 + 캘린더 네비게이션 개선 ⭐
- 효과: 빠른 날짜 이동
- 난이도: 하
- 방법: 캘린더 헤더에 "오늘" 버튼 추가

### [보류] 미용사/서비스 필터
- 1인 미용실이라 당장 불필요
- 직원 추가 시 구현
