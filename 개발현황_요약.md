# 애견미용샵 예약관리 시스템 — 웹 버전 개발 현황

## 한눈에 보기

기존에 PC에서만 사용 가능했던 예약관리 프로그램을 **스마트폰, 태블릿, PC 어디서든 사용할 수 있는 웹 버전**으로 만들었습니다. 인터넷만 되면 어디서든 접속 가능합니다.

---

## 주요 기능

### 1. 캘린더 + 타임라인
- 월간 캘린더에서 날짜별 예약 현황을 한눈에 확인
- 각 날짜에 **반려동물 이름 뱃지**가 표시되어 어떤 아이가 예약되어 있는지 바로 확인
- 날짜를 누르면 **시간대별 타임라인**이 나타남
- PC에서는 왼쪽 캘린더 + 오른쪽 타임라인 **2단 화면**으로 동시 확인 가능

### 2. 예약 등록
- 타임라인에서 **빈 시간대를 터치**하면 바로 예약 등록 가능
- 미용 종류, 털 길이, 소요시간, 금액을 **버튼 터치**로 빠르게 선택
- 이전에 받았던 서비스를 **한 번에 불러오기** 가능 (이전 서비스 이력)

### 3. 예약 수정 / 상태 관리
- 예약을 눌러서 상세 내용 확인, 수정, 메모 추가 가능
- **미용 완료 / 예약 취소 / 노쇼** 처리를 버튼 하나로
- 완료 처리하면 완료 시간이 자동 기록됨

### 4. 예약 날짜/시간 변경
- 예약 상세에서 **"예약 변경"** 버튼을 누르면
- 캘린더에서 새로운 날짜를 고르고, 빈 시간을 터치하면 자동으로 이동 완료
- 화면 상단에 노란색 안내바가 떠서 현재 변경 중임을 알려줌

### 5. 전화 수신 연동
- 매장 전화로 전화가 오면 **자동으로 팝업**이 뜸
- 등록된 고객이면: 반려동물 이름, 견종, 방문 횟수, 최근 이력 표시
- 미등록 번호면: 바로 신규 고객 등록 가능
- 팝업에서 **"예약하기"** 버튼을 누르면 자동으로 예약 등록 화면으로 연결

### 6. 전화 이력 관리
- PC: 왼쪽 하단에 **날짜별 전화 이력** 사이드바
- 모바일: 상단 전화 아이콘을 눌러서 확인
- 이력을 누르면 해당 고객 정보가 팝업으로 표시됨

### 7. 고객 관리
- 고객 검색 (이름, 전화번호, 반려동물 이름으로 검색)
- 고객별 **방문 횟수, 총 매출, 평균 단가** 통계 표시
- 예약 이력을 카드 형태로 한눈에 확인
- 고객 등록 시 견종 **자동완성** 지원

### 8. 보안
- 비밀번호 로그인 (5회 틀리면 10분 잠금)
- "이 기기 기억하기" 체크하면 30일간 자동 로그인
- 모든 데이터는 암호화된 연결(HTTPS)로 전송

### 9. 데이터 백업
- 버튼 하나로 **전체 데이터 JSON 파일 다운로드** 가능
- 고객 정보, 예약 내역, 전화 이력 모두 포함

## 디자인

**토스(Toss) 앱**의 디자인 원칙을 참고하여 제작했습니다.
- 깔끔하고 넓은 여백
- 부드러운 애니메이션
- 중요한 정보가 먼저 눈에 들어오는 구조
- 상태별 색상 구분 (파란색=예약, 초록색=완료, 빨간색=취소, 노란색=노쇼)

## 접속 방법

| 방법 | 설명 |
|------|------|
| **스마트폰** | 브라우저에서 주소 입력 후 "홈 화면에 추가"하면 앱처럼 사용 |
| **PC** | 브라우저에서 주소 입력하여 접속 |
| **태블릿** | 스마트폰과 동일 |

## 기술 요약 (참고용)

| 항목 | 내용 |
|------|------|
| 서버 | Render (클라우드 호스팅) |
| 데이터베이스 | Supabase PostgreSQL (클라우드 DB) |
| 전화 연동 | 안드로이드 Tasker 앱 → 웹훅 방식 |
| 실시간 알림 | SSE (Server-Sent Events) |
