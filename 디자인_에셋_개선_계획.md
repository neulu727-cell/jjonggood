## 1. 현황 분석

### 현재 프로젝트 디자인 상태

| 항목 | 현재 상태 | 비고 |
|------|-----------|------|
| 아이콘 | 인라인 SVG 하드코딩 | 통일성 부족, 유지보수 어려움 |
| 일러스트레이션 | 없음 | 빈 상태(empty state) 등에 활용 가능 |
| 애니메이션 | CSS transition만 사용 | 미세 인터랙션 부재 |
| 로고/브랜딩 | 텍스트 + 🐾 이모지 | 전문적 브랜드 아이덴티티 부재 |
| 폰트 | 시스템 폰트 (Apple SD Gothic Neo, Noto Sans KR) | 적절함 |
| 컬러 시스템 | Toss 스타일 CSS 변수 | 잘 구축되어 있음 |
| 이미지 에셋 | 없음 | 강아지/펫 관련 비주얼 부재 |

### "게임 에셋"과 "웹앱 디자인 에셋"의 차이

게임에는 스프라이트, 텍스처, 3D 모델 등 **게임 에셋**이 존재합니다.
웹앱/생산성 앱에도 이에 대응하는 **디자인 에셋**이 존재하며, 종류는 다음과 같습니다:

| 게임 에셋 | 웹앱 디자인 에셋 | 역할 |
|-----------|------------------|------|
| 스프라이트 시트 | **아이콘 라이브러리** | UI 요소의 시각적 표현 |
| 배경 텍스처 | **일러스트레이션** | 빈 상태, 온보딩, 랜딩 페이지 |
| 이펙트/파티클 | **마이크로 애니메이션 (Lottie)** | 로딩, 성공/실패, 전환 효과 |
| 캐릭터 모델 | **마스코트/브랜드 캐릭터** | 브랜드 아이덴티티, 빈 상태 |
| UI 프레임 | **UI 컴포넌트 킷** | 버튼, 카드, 모달 등 |
| 사운드 에셋 | **알림음** | 예약 알림, 전화 수신 |

## 2. 도입 가능한 디자인 에셋 카테고리

### 2-1. 아이콘 라이브러리 (최우선)

현재 인라인 SVG를 하드코딩하고 있어, 통일된 아이콘 라이브러리 도입이 가장 시급합니다.

| 라이브러리 | 아이콘 수 | 특징 | 적합도 |
|-----------|----------|------|--------|
| **Lucide** | 1,600+ | Feather 포크, 깔끔한 라인 스타일, CDN 지원 | ★★★★★ |
| **Phosphor** | 9,000+ | 6가지 두께(thin~duotone), 다양한 스타일 | ★★★★☆ |
| **Heroicons** | 300+ | Tailwind 제작, 미니멀 | ★★★☆☆ |
| **Tabler Icons** | 5,700+ | 대시보드/관리자 UI 최적화 | ★★★★☆ |

**추천: Lucide Icons**

- 현재 사용 중인 Toss 스타일 디자인과 가장 잘 어울리는 깔끔한 라인 아이콘
- Vanilla JS + CDN으로 바로 사용 가능 (프레임워크 불필요)
- MIT 라이선스, 상업적 사용 가능

```html
<!-- CDN 한 줄로 도입 -->
<script src="https://unpkg.com/lucide@latest"></script>
<script>lucide.createIcons();</script>

<!-- 사용 예시 -->
<i data-lucide="calendar"></i>      <!-- 캘린더 -->
<i data-lucide="scissors"></i>      <!-- 미용 가위 -->
<i data-lucide="phone-incoming"></i> <!-- 수신 전화 -->
<i data-lucide="dog"></i>           <!-- 강아지 -->
<i data-lucide="search"></i>        <!-- 검색 -->
<i data-lucide="user"></i>          <!-- 고객 -->
<i data-lucide="bar-chart-3"></i>   <!-- 매출 차트 -->
```

### 2-2. 일러스트레이션

빈 상태(데이터 없음), 로그인 화면, 에러 페이지 등에 활용할 일러스트레이션입니다.

| 리소스 | 특징 | 가격 | 적합도 |
|--------|------|------|--------|
| **unDraw** | 브랜드 컬러 맞춤, SVG, 테크 스타일 | 무료 | ★★★★☆ |
| **Storyset** | 애니메이션 지원, 다양한 스타일 | 무료 | ★★★★★ |
| **DrawKit** | 세련된 2D/3D 일러스트 | 무료/유료 | ★★★★☆ |
| **Freepik** | 강아지/펫 그루밍 전용 일러스트 풍부 | 무료/유료 | ★★★★★ |

**추천: Storyset + Freepik 조합**

- **Storyset**: 빈 상태, 온보딩용 일반 일러스트 (무료, 컬러 커스텀 가능)
- **Freepik**: 강아지/펫 그루밍 전용 카와이 스타일 일러스트 (출처 표기 시 무료)

**활용 포인트:**
- 예약이 없는 날: "오늘은 한가한 하루!" + 쉬고 있는 강아지 일러스트
- 검색 결과 없음: "검색 결과가 없습니다" + 돋보기 든 강아지
- 로그인 화면: 미용 중인 귀여운 강아지 일러스트
- 첫 사용 안내: 단계별 온보딩 일러스트

### 2-3. 마이크로 애니메이션 (Lottie)

작지만 강력한 인터랙션으로 앱의 품질감을 높입니다.

| 리소스 | 특징 | 가격 |
|--------|------|------|
| **LottieFiles** | 수만 개의 무료 애니메이션, JSON 기반 | 무료/유료 |
| **Lordicon** | 트리거 기반 애니메이션 아이콘 | 무료(50개)/유료 |

**추천: LottieFiles**

```html
<!-- Lottie 플레이어 CDN -->
<script src="https://unpkg.com/@lottiefiles/lottie-player@latest"></script>

<!-- 사용 예시 -->
<lottie-player src="success.json" autoplay loop style="width:120px"></lottie-player>
```

**활용 포인트:**
- 예약 완료 시: 체크마크 애니메이션 (✓ 바운스)
- 로딩 중: 발바닥(🐾) 걸어가는 애니메이션
- 전화 수신: 전화벨 흔들리는 애니메이션
- 매출 페이지: 숫자 카운트업 애니메이션
- 삭제 확인: 부드러운 경고 애니메이션

### 2-4. 마스코트/브랜드 캐릭터

앱의 정체성을 만드는 핵심 요소입니다.

**제안: 귀여운 강아지 마스코트 캐릭터 제작**

- 다양한 감정 표현 (기쁨, 슬픔, 놀람, 졸림 등)
- 각 상태별 활용:
  - 로그인: 반갑게 손 흔드는 강아지
  - 빈 예약: 졸고 있는 강아지
  - 에러: 당황한 강아지
  - 예약 완료: 신난 강아지
- AI 일러스트 생성 도구 또는 Freepik에서 일관된 스타일의 에셋 확보

## 4. 기술 구현 가이드

### 4-1. Lucide Icons 도입 방법

현재 프로젝트는 Vanilla JS + Flask(Jinja2)이므로 CDN 방식이 최적입니다.

**index.html에 추가:**
```html
<!-- head 또는 body 끝에 추가 -->
<script src="https://unpkg.com/lucide@0.460.0/dist/umd/lucide.min.js"></script>
<script>
  // 페이지 로드 후 아이콘 렌더링
  document.addEventListener('DOMContentLoaded', () => {
    lucide.createIcons();
  });
</script>
```

**기존 인라인 SVG 교체 예시:**
```html
<!-- Before (하드코딩) -->
<svg width="18" height="18" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2">
  <circle cx="11" cy="11" r="8"/><line x1="21" y1="21" x2="16.65" y2="16.65"/>
</svg>

<!-- After (Lucide) -->
<i data-lucide="search" style="width:18px; height:18px"></i>
```

### 4-2. 빈 상태 일러스트 적용

```html
<div class="empty-state">
  <img src="/static/assets/no-reservations.svg" alt="예약 없음" width="200">
  <h3>오늘은 예약이 없어요</h3>
  <p>새 예약을 추가해보세요!</p>
</div>
```

```css
.empty-state {
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
  padding: 48px 24px;
  text-align: center;
  color: var(--text-secondary);
}
.empty-state img {
  margin-bottom: 16px;
  opacity: 0.85;
}
.empty-state h3 {
  font-size: 18px;
  font-weight: 600;
  color: var(--text);
  margin-bottom: 8px;
}
```

### 4-3. Lottie 애니메이션 도입

```html
<!-- CDN -->
<script src="https://unpkg.com/@lottiefiles/lottie-player@2/dist/lottie-player.js"></script>

<!-- 로딩 스피너 교체 -->
<lottie-player
  src="/static/assets/paw-loading.json"
  background="transparent"
  speed="1"
  style="width: 60px; height: 60px"
  loop autoplay>
</lottie-player>
```

### 4-4. 발바닥 패턴 배경 (CSS only)

```css
.paw-pattern-bg {
  background-color: var(--bg);
  background-image: url("data:image/svg+xml,..."); /* 발바닥 SVG 패턴 */
  background-size: 40px 40px;
  opacity: 0.03;
}
```

1. **친근감 증가**: 강아지 마스코트와 일러스트로 따뜻한 분위기

3. **인터랙션 만족도**: 마이크로 애니메이션으로 반응성 체감
4. **빈 상태 경험**: 데이터가 없을 때도 즐거운 화면
5. **브랜드 인지도**: 고유한 비주얼 아이덴티티 구축

## 7. 주의사항

- **성능**: 에셋 추가 시 페이지 로드 속도 모니터링 필요
  - Lucide: CDN gzip 시 ~30KB로 가벼움
  - Lottie: JSON 파일 평균 5~20KB
  - 일러스트 SVG: 개당 10~50KB
- **라이선스**: 모든 에셋의 라이선스 확인 필수
  - Lucide, Phosphor, Tabler: MIT (자유 사용)
  - Freepik 무료: 출처 표기 필요
  - unDraw: 완전 무료, 출처 불필요
- **일관성**: 하나의 스타일(라인/솔리드/듀오톤)로 통일
- **접근성**: 아이콘에는 항상 aria-label 또는 대체 텍스트 제공
